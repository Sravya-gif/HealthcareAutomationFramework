# Healthcare Automation Framework

## Project Description

This project is a Selenium Java Automation Framework developed using Page Object Model (POM) for a healthcare appointment booking application.

## Features

* Login and Logout Automation
* Appointment Booking
* Appointment History Verification
* Multiple Appointment Handling
* Form Validations
* Screenshot on Failure
* Extent Reports

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* ExtentReports

## Project Structure
- src/main/java → Framework code
- src/test/java → Test cases
- reports → Extent Reports
- screenshots → Failure screenshots

## Reports

Reports are generated in `/reports` folder.

## Screenshots

Failure screenshots are stored in `/screenshots`.
